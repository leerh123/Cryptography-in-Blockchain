# 区块链中的密码学
使用python对区块链中设计的密码学原语做了实现。除经典实现之外，还实现了常见算法的优化与攻击，具体功能如下   

-    SM3实现及长度扩展攻击

   -   SM3优化使其速度达到3MB/s
   -   48bitSM3生日攻击
   -   FC6962中merkle树的实现
   -   比特币中ECDSA签名伪造
   -   SHA256长度扩展攻击
   -   SM3rho攻击
   -   RFC6962中merkle树的实现
   -   SM4实现及加速
   -   SM2 PGP方案实现
   -   SM2_2P签名方案
   -   SM2_2P解密方案
   -   SM2_RFC6979的实施
